## 🎯 Problem Statement

Fire hazards in homes, industries, and public spaces can escalate rapidly if not addressed immediately. Manual intervention is often delayed and risky.

👉 This project aims to build a **low-cost autonomous robot** capable of detecting and extinguishing fire **without human intervention**.

## 📌 Overview

An intelligent **fire-fighting robot** that:
- Detects flames using IR sensors  
- Navigates autonomously toward the fire  
- Aligns itself using sensor feedback  
- Extinguishes fire using a **targeted water spray system**

This project demonstrates:
- Real-time decision making  
- Sensor-based navigation  
- Embedded system design  
- Hardware-software integration  

---

## 🚀 Key Features

- 🔥 **Automatic Fire Detection** using IR flame sensors  
- 🤖 **Autonomous Movement** based on sensor input  
- 🎯 **Directional Targeting** using servo motor  
- 💧 **Water Pump System** for fire suppression  
- ⚡ **Real-Time Processing** with Arduino Mega  
- 🔄 **Continuous Monitoring** of surroundings
