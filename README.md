## 📌 Overview

This project demonstrates an **autonomous fire-fighting robot** capable of:
- Detecting fire using **flame sensors**
- Navigating toward the fire source
- Extinguishing it using a **water pump + servo mechanism**

It showcases **real-time embedded decision-making** and **hardware-software integration**, making it a strong practical application of robotics in safety systems.

---

## 🚀 Key Features

- 🔥 Automatic Fire Detection using IR flame sensors  
- 🤖 Autonomous Navigation based on sensor input  
- 💧 Water Pump-based extinguishing system  
- 🎯 Servo-controlled directional spraying  
- ⚡ Real-time decision making using Arduino Mega  
- 🔄 Continuous environment scanning
